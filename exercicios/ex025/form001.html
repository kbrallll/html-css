Meu primeiro formulário

Aqui nesse exemplo de formulário estamos criando um formulário com action=cadastro.php que é para onde as informações serão armazenadas e o autocomplete=off que é para quando vc digitar o texto na caixa do formulário não se auto completar!!!
Com inputtype=text e inputtype=submit para enviar o formulário!!!
A label é para etiquetar a caixa de formulário para javascript e label o melhor a se usar é ID, para PHP e Pyton o melhor é o name
O atributo method="POST" na tag Label é para esconder as informações da URL
O atributo method="GET" é o metodo padrão e os dados aparecem na URL e tbm quando vc quiser compartilhar a URL é a melhor maneira...
